-------- spatial statistics functions -------- 

 
lmerror    : LM error statistic for spatial correlation in residuals
lmerror_d  : An example of using lmerror()
lmsar      : LM error statistic for spatial correlation in residuals
lmsar_d    : An example of using lmsar()
lratios    : computes likelihood ratio test for spatial correlation
lratios_d  : An example of using lratios()
moran      : computes Moran's I-statistic for spatial correlation
moran_d    : An example of using moran()
walds      : Wald statistic for spatial autocorrelation in residuals
walds_d    : An example of using walds()
