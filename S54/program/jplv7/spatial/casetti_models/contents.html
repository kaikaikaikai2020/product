-------- Casetti model functions -------- 

 
bcasetti    : computes a heteroscedastic Bayesian variant of Casetti's spatial expansion regression
bcasetti_d  : An example of using bcasetti
casetti     : computes Casetti's spatial expansion regression
casetti_d   : An example of using casetti
darp        : computes Casetti's DARP model
darp_d      : An example of using darp
darp_lik1   : evaluate the log-likelihood for the DARP model
darp_lik2   : evaluate the log-likelihood for the DARP model
prt_cas     : Prints output using spatial expansion results structures
