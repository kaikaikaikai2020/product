-------- Spatial Panel data functions -------- 

 
demosarfe     Demonstrates SAR model with Fixed Effects
demosemfe     Demonstrates SEM model with Fixed Effects
f2_sarpanel  : evaluates log-likelihood -- given ML estimates
f2_sempanel  : evaluates log-likelihood -- given ML parameters
f_sarpanel   : evaluates concentrated log-likelihood for the sar
f_sempanel   : evaluates concentrated log-likelihood for the sem
prt_sp       : Prints output using spatial panel regression results structures
sar_panel    : computes spatial lag model estimates for spatial panels (N regions*T time periods)
sem_panel    : computes spatial error model estimates for spatial panels (N regions*T time periods)
